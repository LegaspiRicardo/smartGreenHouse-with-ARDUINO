Smart green house able to grow plants inside. Controlled with an automatic system of water irrigation, ventilation and lighting.

It was made in C++

Components that I used.
- Arduino MEGA
- I2c Module
- Humidity Sensor (DHT22) 
- Relay module
- 4 Fans
- Display (LCD 16*2)
- Clock (RTC DS-3231)
- 2 Water Pumps
- UV & Infrared Light
